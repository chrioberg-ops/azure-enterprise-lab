# Evidence

This folder contains screenshots and technical evidence collected during the Azure Enterprise Lab project.

## Structure

- `01-azure-infrastructure` – Resource group, virtual network, subnets and virtual machines
- `02-network-security` – Network Security Groups, RDP restrictions and private access
- `03-active-directory` – Domain, OUs, users, groups and DNS
- `04-group-policy` – GPO configuration and policy validation
- `05-file-services` – Department shares, NTFS permissions, SMB permissions and access tests
- `06-backup` – Recovery Services Vault, backup jobs and recovery points
- `07-monitoring` – Log Analytics, Azure Monitor, DCR, collected data and alerts
- `08-powershell` – Script execution and idempotency validation
- `09-intune-limitation` – Tenant, role and licensing limitation evidence

## Naming Standard

Use descriptive filenames with a number and short title.
