# Azure Enterprise Lab

## Project Overview

Azure Enterprise Lab is a vocational graduation project focused on designing, implementing, and documenting a modern enterprise IT environment in Microsoft Azure.

The project is based on a fictional company, **Nordic IT Solutions AB**, a growing IT consulting company with approximately 50 employees working both on-site and remotely.

The goal is to demonstrate practical skills in:

* Microsoft Azure
* Windows Server Administration
* Active Directory
* Microsoft Entra ID
* Microsoft Intune
* Network Design
* PowerShell Automation
* Monitoring and Logging
* Backup and Recovery
* Technical Documentation

---

## Business Scenario

Nordic IT Solutions AB is expanding and requires a modern cloud-based infrastructure to centrally manage users, devices, security policies, monitoring, and backup operations.

As the System Administrator, the objective is to design and implement a secure and scalable environment that supports both office-based employees and remote consultants.

---

## Project Goal

The purpose of this project is to investigate how Microsoft Azure can be used to build, secure, automate, and administer a modern enterprise IT environment.

---

## Project Roadmap

* Define business requirements

* Create project documentation

* Deploy Azure resources

### Phase 3 – Identity Management

* Deploy Active Directory

### Phase 4 – Device Management

* Enroll Windows devices

### Phase 5 – Automation

* Automate administrative tasks

* Test recovery procedures

* Final documentation
* Lessons learned
